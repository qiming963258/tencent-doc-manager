## 🔧 完整处理流程

### 10步核心流程
```
1. 基线设定 → 2. 获取变更 → 3. 初步对比 → 4. AI列名标准化 → 5. 数据清洗打分
     ↓
6. UI参数生成 → 7. 热力图显示 → 8. Excel半填充 → 9. 上传文档 → 10. UI链接绑定
```
